# GENERATIVE_AI_PROJECT
The intention behind this project is to replicate an actual interview session between human and AI. There has been a lack of dynamism and flexibility within existing interview methods for the assessment of candidates' knowledge and competencies. With this project, the aim is to introduce an advanced form of an adaptive interviewing technique powered by the Gemini 3.1 Flash Lite Preview model. I use the "Knowledge Map" concept in which I allow the AI to adjust the difficulty of questions and their topic relevance based on the performance of the candidate at any point during the session. In doing so, I have considered the interview as a problem of optimal assessment by treating it as an optimal solution of a functional space. Consequently, it helps minimize the chances of any form of bias and provides information on the technical abilities of candidates. This project seeks to provide an alternative to conventional interviewing techniques, where each candidate is judged using the same set of questions. LLMs are used to provide candidates with a conversational interface to answer questions on various technical subjects like machine learning, data structures, and system design among others.
In this project, the relationship between aleatory uncertainty (randomness in the candidate's communication) and epistemic uncertainty (uncertainty in the system due to unknown ability of the candidate) has been considered.
The concept behind this project was the annoyance with repeated technical assessments which don't adapt to the candidate's responses. Therefore, our aim was to develop an application which felt like a dialogue with a senior software engineer where the difficulty of the questions increases as we answer correctly and decreases when we stumble upon any point.
